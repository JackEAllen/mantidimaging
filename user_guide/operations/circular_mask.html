Circular Mask¶

Important

This filter requires TomoPy to be available.

Applies TomoPy’s circular mask filter (tomopy.misc.corr.circ_mask) (docs).

Takes a radius as a ratio of the smallest image dimension and a value to apply to all pixels outside this radius.
