class mantidimaging.gui.windows.savu_operations.process_list.view.PluginDisplayWidget(parent: PyQt5.QtWidgets.QWidget, pos: Union[int, str], plugin_name: str, fixed: bool = False)[source]¶

Bases: PyQt5.QtWidgets.QWidget

Widget representing a plugin in the process list.

Each widget is laid out horizontally and consists (in order) of:

  1. A label with the plugins position in the process list

  2. A button with the plugins name. Clicking this makes the parameters of the plugin editable.

  3. Up/Down arrows to change the plugins position in the process list

  4. A remove button.

All elements except the plugin button are fixed size, taking up as little (horizontal) space as possible. The plugin button fills the remaining space, which will be determined by the width of the largest name of any plugin in the parent layout.

The view of the process list in the savu operations window.

A fixed loader and saver plugin are represented by non-interactive widgets. Processing plugins are laid out in between, and can be edited, removed, and have their positions changed.
